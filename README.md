# 🧠 AI Resume Analyzer

An AI-powered web app that analyzes resumes against job descriptions and provides match scores, missing skills, and improvement suggestions.

# App Link : https://ai-resume-analyzer-jhgnwhagwgxlex4jrrr7zy.streamlit.app/

## 🚀 Features

* Resume parsing (PDF)
* Skill extraction (technical + non-technical)
* Match score using NLP (BERT)
* Role detection (Data Science, Web Dev, Marketing, etc.)
* Skill gap analysis
* Smart suggestions

## 🛠️ Tech Stack

* Python
* Streamlit
* NLP (Sentence Transformers)
* PyPDF2

## 📊 How it works

1. Upload your resume
2. Paste job description
3. Get:

   * Match score
   * Missing skills
   * Suggestions

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
